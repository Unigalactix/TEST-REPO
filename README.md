# jira-agent-sample-node

Minimal Express app with Jest tests, suitable for CI pipelines.

## Scripts
- `npm run build`: No-op build step for CI compatibility
- `npm test`: Runs Jest tests
- `npm start`: Starts the server on port 3000

## Try it locally
```powershell
# Install dependencies
npm install

# Run tests
npm test

# Start the app
npm start
```

Visit http://localhost:3000/health
